SQL Server JSON

In and Out

JSON

A way to serialize data, including hierarchical data, into a string format.

Similar to what we do with XML, but a slightly simpler format.

JSON

{ } defines an object. Contains one or more properties each of which can be arbitrarily complex.

[ ] defines an array. Multiple occurences of objects

"name": "value" defines a property (a "field") of the object. If the value is numeric it is not enclosed in quotes.

Sample

	
  {
    "menu": {
      "id": "myitem",
      "value": "My Item",
      "sequence": 37,
      "popup": [
        {
          "value": "Open",
          "onclick": "OpenDoc()"
        },
        {
          "value": "Close",
          "onclick": "CloseDoc()"
        }
      ]
    }
  }

Generating JSON

  • The result set of query can be requested to be a JSON string
  • The entire result set becomes a JSON string

Pulling Data From JSON

  • JSON_VALUE pulls a single scalar value from a JSON string
  • JSON_QUERY pulls a subset of the JSON whcih is itself a JSON string
